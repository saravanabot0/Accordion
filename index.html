What is HTML ?

HTML stands for Hypertext Markup Language and is the standard markup language used to create web pages. It's the most basic building block of the web and is used to structure web pages, add visual formatting, and make them accessible to users.

What is CSS ?

Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

What is JS ?

JavaScript is a lightweight, cross-platform, single-threaded, and interpreted compiled programming language. It is also known as the scripting language for webpages. It is well-known for the development of web pages, and many non-browser environments also use it.
